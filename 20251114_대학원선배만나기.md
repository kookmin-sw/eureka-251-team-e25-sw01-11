# 대학원 선배 만나기 - 20251114

&nbsp;

## 데이터마이닝 랩 박기연 선배님

&nbsp;

**Q. 대학원에 대한 편견, 밤샘 연구나 고강도 노동은 정말인가요?**

A. 연구실마다 다르지만, DM Lab은 그런 문화가 없고 비교적 분위기가 좋다.

&nbsp;

**Q. 대학원의 장점이 있다면?**

A. 연구실에 들어가면 매달 장학금 혹은 인건비 형태로 일정 금액을 받는다. 월급처럼 꽤 쏠쏠하다.

&nbsp;

**Q. 대학원에 진학하려면 어떻게 해야하나요?**

A. 학점은 절대적으로 중요한 요소는 아니다. 타대 출신도 충분히 받아주고, 미리 경험해보고 싶다면 학부 3~4학년 때 대학원 강의를 미리 수강해보는 것도 좋은 방법이다.

&nbsp;

**Q. 인공지능, 데이터 마이닝의 비전은 어떨까요?**

A. 인공지능 취업 관련해서는 경량화, RAG 모델, 비전모델, 비전랭귀지 등이 미래가 밝다.

데이터 마이닝은 단순히 데이터 분석처럼 결과를 설명하는 데 집중하는 것이 아니라, 데이터 내부에서 패턴이나 유사성을 발견하는 데 초점이 있다.

또한 요즘은 데이터 마이닝 분야에서도 LLM 경량화 같은 모델 관련 연구까지 다루는 경우가 많다.

&nbsp;

**Q. 졸업 논문 주제를 여쭤봐도 되나요?**

A. 날씨 데이터에서 유사성 발견하기를 주제로 하고있다. 데이터 수집은 이전 연구에서 사용되던 데이터를 이용하거나, 국가기관에 직접 요청하면 수집할 수 있다.
